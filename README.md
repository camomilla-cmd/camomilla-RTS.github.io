# App 6 GitHub Pages Site

This folder is ready to upload to an existing GitHub repository for GitHub Pages.

## Included Files

- `index.html` - the main project webpage
- `styles.css` - styling for the page
- `application6.mp4` - local demo video preview
- `App6` - project source file

## Before Publishing

1. Upload your demo video to YouTube.
2. Open `index.html`.
3. Replace `YOUTUBE_VIDEO_ID` with your real YouTube video ID.

Example:

- YouTube URL: `https://www.youtube.com/watch?v=abc123xyz`
- Embed ID: `abc123xyz`

## Publishing with GitHub Pages

In your GitHub repository settings:

1. Open **Pages**.
2. Set the source to the branch you want to publish from.
3. Choose the folder that contains `index.html`.
4. Save the settings and wait for the site URL to become available.
